About the Lazarus Program

The LAZARUS project is multidiscipline project hosted by the math department at Texas Tech University with the goal of first increasing interest in STEM(science, technology, engineering, mathematics) disciplines in high school and undergraduate students second providing research opportunism to undergraduate students that would normally not be available. These goals will be met by building a GPU supercomputing cluster. This cluster will first be used to run disease, specifically zombie, simulations.
