Detailed Description

template<typename T, typename L>
struct allocation_policy< T, L >

Just a wrapper class to make a single interface to the allocator types.

for locations device,pinned,managed the implementation is a buddy block allocator. host location the default is the standard allocator.

Parameters
Ttype being allocated by allocator
Llocation policy. one of host,device,pinned,managed if L is host and T is not given then T defaults to standard_alloc_p
//to insitiate a allocater with buddy block strategy on the device and then
*allocate
typedef allocation_policy<int,device> Allocator;
Allocator allocator;
Allocator::pointer p=allocator.allocatoe(sizeof(int)*2);
